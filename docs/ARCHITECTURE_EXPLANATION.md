
# FITI Architecture Explanation

## Multi-Agent System
FITI now separates intelligence into multiple agents.

Benefits:
- cleaner reasoning
- better scalability
- easier debugging
- specialized intelligence

## Streaming UX
The system streams live reasoning to the frontend.

Benefits:
- modern AI-native experience
- real-time feedback
- premium interaction feel

## Vector Memory
Stores semantic financial memory.

Benefits:
- long-term personalization
- recurring pattern understanding
- historical financial intelligence

## LangGraph
Future orchestration layer.

Benefits:
- stateful workflows
- multi-agent coordination
- advanced reasoning systems
